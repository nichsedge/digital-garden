# Projects

A short list of things I’ve built or contributed to. If something here interests you, I’m happy to walk through the architecture and tradeoffs.

## [sans-finance](https://github.com/nichsedge/sansfinance)

Android personal finance + wealth app (Kotlin/Compose, Clean Architecture). Privacy-first with on-device AI insights and receipt scanning via LiteRT-LM.

## [portfolio-integration](https://github.com/nichsedge/portfolio-integration)

Python/Node monorepo ETL that integrates portfolio data (KSEI, Binance, DeBank, Hyperliquid, etc.) + manual CSV into unified daily snapshots and charts.

## [idx-bei](https://github.com/nichsedge/idx-bei)

Indonesia Stock Exchange (IDX/BEI) data toolkit: scraping + processing + analysis (optional Neo4j ingestion) for market & company fundamentals.

## [k8s-ansible-gcp](https://github.com/nichsedge/k8s-ansible-gcp)

Infrastructure automation on GCP using Kubernetes + Ansible (provisioning and ops patterns).

## [indoscraping](https://github.com/nichsedge/indoscraping)

Collection of web scrapers for Indonesian news portals (Detik, Kompas, CNBC, CNN) and retail sites (Alfagift, Indomaret) in Python and Node.js.

## [social-media-analytics](https://github.com/nichsedge/social-media-analytics)

Social media analytics project repo: data collection, text mining, topic modeling, stance detection, and network analysis (PageRank).
